Macspell

About

MacSpell is a spell checker designed on Cocoa's spell-checking facilities. MacSpell can function as an ispell -a replacement.

Install

curl -O https://raw.github.com/ruda/macspell/master/macspell.py
chmod +x macspell.py

Usage

Check spelling of README file.

./macspell.py --list < README

Learn new words.

./macspell.py --learn
Rudá
Sumé
^D

Forget learned words:

./macspell.py --unlearn
Rudá
Sumé
^D

Emacs and MacSpell

Include in your initialization file:

;; MacSpell
(setq ispell-program-name "/path/to/macspell.py")
(setq ispell-extra-args '("--encoding=latin1" "--auto-lang=no"))
;(setq ispell-dictionary "english")
